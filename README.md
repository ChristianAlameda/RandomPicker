# RandomPicker
A program to decide what essay prompt I should write for class. Ended up adding sound effects, music, and photos. Hope you enjoy
Created this program for essay ideas that I needed to have for my english class. I needed a different essay prompt every day for a hundred days
so I figured it would be fun to make something that allowed me to not think about which topic but rather have the topic given to me. 

The program will give you a random topic based on what is in the data base and it will remember what topic you chose for that day add it to the finished essay 
prompt list and delete it from the possible essay prompt list. That way you don't have to remember what you wrote about on Tuesday. You can simply go to printlist 
press 1 and you'll be able to see the list of possible topics as well as the finished topics you have done already. 

There is also a scandolous option in the printlist options which will give 10 warning quotes from various movies, books, or plays and give you the option to continue
or flee, if you continue you will be bombarded with pictures of dogs with hats with fun music in the background. It will take about 5 minutes for it to end, so continue
as you will
